1. (< h > < / h >)

H1


h diatas biasanya disebut dengan heading yang memiliki h1 sampai dengan h6, yang bermaksud h1 yaitu ukuran yang paling besar dan h 6 yaitu ukuran paling kecil dan biasanya heading ini di pergunakan untuk membuat artikel/blog

2.< p >

paragraf

tag p ini untuk membuat paragraf

3. < br >
tag br ini untuk membuat garis baru

4. < a > Menuju ke youtube
tag a ini digunakan untuk menyimpan link bahwa link ini ketika di klick akan di tampilkan di halaman web baru atau menggantinya dengan web saat ini

5. < head >
tag head ini biasanya berisikan dokumen dokumen tentang html tag ini tidak akan tampil dalam website

6. < tittle >
tag tittle ini digunakan untuk memberi judul laman website

7. < table >
tag table ini digunakan untuk membuat table

8. < t body >
tag tbody ini digunakan untuk membungkus bagian body dari table

9. < thead >
tag thead digunakan untuk membungkus kepala bagian table

10. < img >
tag img digunakan untuk menampilkan gambar pada halaman web

11. < mark >
tag mark ini digunakan untuk menunjukan sebuah teks dalam dokumen yang di ditandai atau disorot terhadap konteksnya. contohnya seperti ini = dokumen

12. < font >
tag ini digunakan untuk mendifinisikan jenis/ style font pada tulisan yang nanti akan di inputkan.

13. < ol >
order list ini untuk membuat list yang di dalam ol ini akan dimasukan li yaitu list.

14. < hr >


tag ini biasanya digunakan untuk memisahkan konten dengan menampilkan garis pembatas

15. < background color >
berfungsi untuk memberikan warna pada latar belakang website

16. < b >
tag di atas berfungsi untuk menebalkan huruf b ini singkatan dari bold contohnya < b > Berfungsi < / b > = Berfungsi

17. < strong >
membuat teks penting sama seperti bold < b > contohnya < strong> Jagung < / strong > = Jagung

18. < tr >
tr = table row berfungsi untuk membuat baris pada table.

19. < id >
id ini memberikan id pada table atau kolom

20. < audio >
tag ini untuk membuat audio player lalu kita bisa mengisikan audio dengan tag < source >.
